# Assumption Rules

- Never invent data
- Clearly label assumptions
- Use conservative estimates by default
- If churn unknown → assume 5–10% monthly for early startups
- If CAC unknown → estimate based on channel benchmarks
- Always explain why an assumption is chosen

## Scenario Rules

### Conservative
- Lower growth
- Higher churn
- Higher CAC

### Base Case
- Realistic expected numbers

### Aggressive
- Higher growth
- Better retention
- Lower CAC
